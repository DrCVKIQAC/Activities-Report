## Features
- 📊 Overview with KPI cards, charts, monthly trends
- 📤 Google Form integration for faculty submissions
- 📋 Full activity register with search, filter, sort
- 📈 Analytics: year-wise, department-wise, type-wise
- 🏛️ Department view cards
- 📅 Year-wise collapsible reports
- 🏆 NAAC Snapshot with live metrics
- 📄 **Auto PDF Report Generation** with geo-tagged photos
- 📧 **Email to faculty in-charge + IQAC** built-in
- 📥 CSV import from Google Sheets
